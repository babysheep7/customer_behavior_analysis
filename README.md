# Customer Shopping Behavior Analysis

## Overview
An end-to-end data analytics project analyzing customer shopping behavior from 3,900 transactions. The project uses Python, PostgreSQL, SQL, and Power BI to transform raw data into actionable business insights.

## Technologies
- Python 
- PostgreSQL
- SQL
- Power BI

## Skills Demonstrated
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Relational Database Management
- SQL Query Development
- Customer Segmentation
- Business Intelligence & Dashboard Design
- Data-Driven Decision Making

## Key Analysis
- Revenue analysis by gender and age group
- Subscriber vs. non-subscriber comparison
- Customer segmentation (New, Returning, Loyal)
- Top-rated and best-selling products
- Discount effectiveness evaluation
- Shipping preference analysis
- Purchase behavior insights

## Dashboard
Developed an interactive Power BI dashboard to monitor:
- Revenue performance
- Customer segments
- Product trends
- Subscription behavior
- Shopping patterns

## Key Outcomes
- Identified high-value customer segments
- Evaluated the impact of discounts and subscriptions
- Highlighted top-performing products and categories
- Provided insights to support marketing and retention strategies
